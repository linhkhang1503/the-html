1. HTML là gì?

HTML (HyperText Markup Language) không phải là ngôn ngữ lập trình, mà là ngôn ngữ đánh dấu dùng để mô tả cấu trúc của một trang web.
Trình duyệt (Chrome, Firefox...) đọc file HTML và hiển thị thành giao diện mà bạn thấy.

-- phân cách dòng
Ba trụ cột của một trang web:
HTML → cấu trúc / nội dung (bộ khung)
CSS → giao diện / trang trí (màu sắc, bố cục) JavaScript → hành vi / tương tác (xử lý logic)

HTML gồm các thẻ (tag), thường đi theo cặp: thẻ mở và thẻ đóng . Nội dung nằm giữa hai thẻ đó gọi là phần tử (element).
